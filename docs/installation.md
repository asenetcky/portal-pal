# Installation

## From source

The source files for Open Data Portal Pal can be downloaded from the [Github repo](https://github.com/asenetcky/portal-pal).

You can either clone the public repository:

```sh
git clone https://github.com/asenetcky/portal-pal
```

Or download the [tarball](https://github.com/asenetcky/portal-pal/tarball/main):

```sh
curl -OJL https://github.com/asenetcky/portal-pal/tarball/main
```

Once you have a copy of the source, you can install it with:

```sh
cd portal-pal
uv sync
```
